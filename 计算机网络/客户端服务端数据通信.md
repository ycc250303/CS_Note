
# 客户端发送 "Hello World" 到服务器的完整网络流程

本文从 **客户端应用 → 操作系统 → 协议栈 → 网络设备 → 服务器协议栈 → 服务器应用** 全链路说明一次网络请求的数据传输过程。

# 一、整体流程概览

```
客户端应用
↓
Socket API
↓
TCP 协议
↓
IP 协议
↓
数据链路层（Ethernet）
↓
网卡
↓
交换机 / 路由器
↓
服务器网卡
↓
服务器协议栈
↓
服务器应用
```

# 二、客户端应用层

示例代码：

```java
Socket socket = new Socket("server_ip", 8080);
OutputStream out = socket.getOutputStream();
out.write("Hello World".getBytes());
```

* Java 将字符串 `"Hello World"` 转换为字节数组
  * `48 65 6c 6c 6f 20 57 6f 72 6c 64`
  * 长度：11 bytes
* 调用 `write()` 发送数据

# 三、系统调用（用户态 → 内核态）

* 应用调用：`write(socket_fd, buffer, 11)`
* 发生：`用户态 → 内核态`
* 操作系统开始处理数据发送。

# 四、Socket 层

内核找到对应的 socket 结构：

```
进程
 │
 ├── fd table
 │      │
 │      └── socket fd
 │
 └── socket struct
```

socket 内部包含：

```
发送缓冲区（send buffer）
接收缓冲区（recv buffer）
TCP状态
对端地址
```

数据被拷贝到发送缓冲区

# 五、TCP 层（传输层）

TCP 将数据封装为  **TCP Segment** 。`TCP Header + Data`。TCP 负责

```
序列号
确认应答
重传
滑动窗口
拥塞控制
```

# 六、IP 层（网络层）

TCP Segment 封装为  **IP Packet** ：`IP Header + TCP Segment`

```
Source IP: 192.168.1.10
Dest IP:   192.168.1.20
Protocol: 6 (TCP)
```

# 七、数据链路层（Ethernet）

IP Packet 被封装为  **Ethernet Frame** 

# 八、ARP 地址解析（如果需要）

如果客户端不知道服务器 MAC 地址，会触发  **ARP 协议** 。

# 九、网卡驱动

数据进入网络驱动层：

```
Socket Buffer
   ↓
sk_buff
   ↓
NIC Driver
```

Linux 内核使用 `sk_buff` 表示网络包。

# 十、DMA（Direct Memory Access）

网卡使用 **DMA** 直接访问内存

CPU 只负责告诉网卡：数据地址和数据长度，减少 CPU 负担

# 十一、物理层发送

网卡将数据转换为：电信号（网线）/光信号（光纤）/无线信号（WiFi）。发送到网络

# 十二、网络设备转发

## 交换机

转发数据帧

## 路由器

转发 IP 包。每经过一个路由器：TTL - 1，并重新封装 MAC 地址

# 十三、服务器网卡接收

服务器网卡接收信号：电信号 → 二进制数据

进行 CRC 校验，并验证数据正确性

使用 DMA 将数据写入服务器内存

# 十四、服务器协议栈处理

服务器内核逐层解析数据

* 数据链路层检查 MAC 地址
* IP 层检查 IP 地址
* TCP 层检查端口，找到对应 socket
* 数据放入缓冲区

# 十五、服务器应用层

```java
InputStream in = socket.getInputStream();
in.read(buffer);
```

调用 read，从内核态转为用户态，将 socket buffer 拷贝到 用户 buffer，得到最终数据

# 十六、完整数据流

```
客户端应用
   ↓
write()
   ↓
用户态 → 内核态
   ↓
Socket Send Buffer
   ↓
TCP Segment
   ↓
IP Packet
   ↓
Ethernet Frame
   ↓
网卡 DMA
   ↓
网络传输
   ↓
服务器网卡 DMA
   ↓
Ethernet
   ↓
IP
   ↓
TCP
   ↓
Socket Receive Buffer
   ↓
read()
   ↓
内核态 → 用户态
   ↓
服务器应用
```
